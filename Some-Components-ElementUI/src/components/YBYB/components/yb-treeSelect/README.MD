## Props

| 参数名           | 类型    | 用途                   | 示例    | 默认值  |
| ---------------- | ------- | ---------------------- | ------- | ------- |
| defaultProps     | Object  | el-tree的结构 替换键名 |         |         |
| data             | Array   | el-tree的数据          |         |         |
| maxHeight        | String  | dropDown的高度         | “200px” | “200px” |
| defaultexpandall | Boolean | 是否默认翟凯所有子节点 |         | true    |
| filterable       | Boolean | 是否显示搜索框         |         | false   |
|                  |         |                        |         |         |
